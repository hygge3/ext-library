# ext 验证码

## 使用

### maven

```xml

<dependency>
    <groupId>ext.library</groupId>
    <artifactId>ext-json</artifactId>
    <version>${version}</version>
</dependency>
```

### gradle

```groovy
compile("ext.library:ext-json:${version}")
```
